HTML (HyperText Markup Language) adalah bahasa standar yang digunakan untuk membuat dan menyusun halaman web. HTML memungkinkan kita untuk menambahkan teks, gambar, video, tautan, dan elemen-elemen lainnya dalam sebuah halaman web.

üõ†Ô∏è Fungsi HTML

  • Membuat struktur dasar halaman web
  • Menentukan elemen seperti heading, paragraf, dan daftar
  • Menambahkan gambar dan multimedia
  • Membuat tautan antar halaman atau situs

üìÑ Contoh Kode HTML Sederhana

<!DOCTYPE html>
<html>
<head>
    <title>Halaman Pertama Saya</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama saya.</p>
</body>
</html>
